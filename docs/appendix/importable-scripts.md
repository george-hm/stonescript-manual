# Importable Scripts

Stone Story RPG ships with a library of ready-to-use Stonescript modules. Use the [`import`](/guide/importing) command to load them.

The official scripts are hosted at `https://StonestoryRPG.com/stonescript/` and are also available in your local save-file folder under `/Stonescript/`.

## Standalone scripts

```stonescript
import Chisel
import CozyCave
import Fishing
import TrainAdventure
```
