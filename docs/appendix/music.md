# Music

Stonescript can play game music tracks using [`music.Play()`](/reference/native-functions/music).

Available tracks depend on the platform. There can only be one track playing at a time.

## Example

```stonescript
?loc.begin | loc.loop
  music.Play(temple_0)
```
